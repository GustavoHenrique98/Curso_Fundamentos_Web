Tags para destaque de texto

Tags < i > e < em > => (itálico)

A tag < i > é usada apenas para marcar visualmente uma palavra em itálico. Já a tag < em > aplica também um itálico, só que esta tag ela funciona para dar uma ênfase semântica em um palavra ou texto . Exemplo : Eu amo Lasanha!

Tags < b > e < strong > => (negrito)

A diferença da tag < b > para tag < strong > é que a tag < b > serve apenas para destacar um texto em negrito sem significado semântico nenhum, já a tag < strong > ela marca o texto como negrito e além disso insere um signifcado semântico de algo importante para o motor de busca do navegador como por exemplo:

Links (âncoras)

A tag < a > serve para criarmos links na nossa página que podem apontar tanto para um endereço de um site, tanto para documentos HTML , ou seções.

Exemplos:

< a > href="http://www.google.com" target="_blank"> Link apontado para o site da google < /a >
< a > href="06-Resumo_do_conteudo.html" target="_blank">Link apontado para este documento HTML < /a >
< a > href="#secao" target="_blank">Link apontado para uma seção criada dentro do documento HTML < /a >

Meta tags

As meta tags em HTML são elementos que fornecem metadados sobre uma página web. Eles são inseridos dentro da seção head do documento HTML e não são visíveis para os usuários. Em vez disso, as meta tags são usadas por navegadores, motores de busca e outras ferramentas para obter informações sobre a página.

Exemplos de meta tags

  • < meta name="viewport" width="device-width, initial-scale=1.0> Configurações para tornar a página responsiva
  • < meta name="descripition" content="Uma descrição breve e concisa da página">
  • < meta name="keywords" content="resumo do conteúdo da seçã 01" >
  • < meta charset="UTF-8" > : Define a codificação de caracteres para a página utilizando UTF-8
