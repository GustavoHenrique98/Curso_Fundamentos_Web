Destaques em textos

Texto em Itálico com i - serve apenas para alterar o visual do texto para itálico Texto em negrito com b - serve apenas para destacar o texto em negrito visualmente
Texto em itálico com em - serve para alterar o texto para itálico e aplicar uma carga semântica de ênfase. Texto em negrito com strong - serve para destacar o texto com negrito e aplicar uma carga semântica de importância

A tag MARK serve para DESTACAR UM TEXTO - não tem significado semnântico para o browser serve apenas para fazer uma marcação de marca texto mais personalizada...
